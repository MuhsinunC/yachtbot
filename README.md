# discord-crypto-webhook

## How to run

### Prerequisites

1. Node
2. NPM or Yarn
3. Docker (optional)

### Traditional

```sh
$ git clone https://github.com/YACHTPARTY/yachtbot.git
$ cd discord-crypto-webhook
$ npm install
$ npm start
```

### Docker

```sh
$ docker run -d syednashikaman/discord-crypto-webhook:latest
```
